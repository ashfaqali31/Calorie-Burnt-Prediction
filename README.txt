Calorie Prediction Project

Overview:
This project predicts calorie expenditure using machine learning models, including Linear Regression, Lasso, ElasticNet, Random Forest, and a Neural Network. The repository contains the Jupyter Notebook used for data preprocessing, visualization, model building, and evaluation.

Contents:
- Calorie Prediction Final.ipynb : Main analysis and modeling notebook
- requirements.txt : Python dependencies needed to run the project
- data/ : (Optional) Place your dataset here if needed

How to Run:
1. Install dependencies:
   pip install -r requirements.txt
2. Launch Jupyter:
   jupyter notebook
3. Open the notebook:
   Calorie Prediction Final.ipynb

Libraries Used:
Pandas, NumPy, Matplotlib, Seaborn, Plotly, Scikit-Learn, TensorFlow.
